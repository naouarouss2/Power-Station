I really love robotics, but the problem I always encounter when prototyping, is how to power on my circuits.

Previously, I used some 18650 batteries and a buck converter, but this is a pretty painful and time-consuming process.

And since my friend gave me an old ATX power supply, I decided to turn it into my own, custom built, lab bench power supply, which I called **Power Station**
## Features âœ…
-  350W total power output
-  3.3V Output
-  5V Outputs
-  12V Outputs
-  A 150W 0.5-40V @ 0-8A variable output
-  A Bluetooth interface for the variable power output
-  A small TFT screen to show some graphs and values regarding the variable power output and battery charging.
-  XT60 Male connector to charge lithium batteries
-  Protection fuses
